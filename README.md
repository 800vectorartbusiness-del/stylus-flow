# Creative Stylus Webapp 🎨

A whimsical pastel‑UI drawing app built with HTML, CSS, and JavaScript.  
Designed for Creative Bharat ecosystem — lets you draw with touch, undo/redo strokes, change brush size, pick colors, clear canvas, and export drawings.

---

## ✨ Features
- 🎨 Gradient pastel background with smooth animation
- 🖌️ Touch‑based drawing on canvas
- ↩️ Undo / ↪️ Redo support
- ➕ / ➖ Brush size control
- 🎨 Custom color picker
- 🗑️ Clear canvas instantly
- 💾 Export drawing as PNG
- 🔗 QR code + WebSocket connect for live sync
